# points-regulator

## Примечание
Узкое и костыльное решение, работающее только для 4-х точек

## Использование
1. Клонируйте репозиторий на локальную машину
2. Запустите сервер
```bash
  make run-server
```
3. Запустите клиент
```bash
  make run-client
```
4. Поменяйте значения в файле assets/input.txt
